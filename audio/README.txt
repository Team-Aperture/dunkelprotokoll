Optional audio for "Dunkelprotokoll".

The game is fully playable SILENT. Drop any of these .mp3 files into this folder
and they light up automatically — each one is optional, and a missing file is
harmless (no errors, the sound toggle still works). Toggle sound with the
🔊 / 🔈 button (bottom-left).

LOOPS (start/stop with the sound toggle):
  music.mp3        background music
  storm.mp3        rain / storm ambience
  vtgm-hum.mp3     V-TGM proximity hum — its volume rises as she gets closer

ONE-SHOTS (triggered by events):
  thunder.mp3      lightning strike — plays on every flash (synced to the storm)
  vtgm-alert.mp3   V-TGM notices you and starts hunting
  caught.mp3       caught by the V-TGM (reset to checkpoint)
  step.mp3         a footstep on each move (keep subtle, or omit the file)
  bump.mp3         walking into a wall
  button.mp3       pressing a wall button
  door.mp3         a door opening
  page.mp3         collecting a page
  checkpoint.mp3   reaching a checkpoint
  ladder.mp3       using a ladder (changing floor)
  win.mp3          protocol complete (coordinates released)

Want more? Easy to add named slots for, e.g.: random metal creaks / water drips,
a heartbeat while hunted, a safe-refuge chime, a coordinate-reveal shimmer, or
menu UI clicks. Just ask and tell me the filename.

No audio files are committed, to keep the repo asset-free.
