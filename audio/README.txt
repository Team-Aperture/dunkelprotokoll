Optional audio for "Dunkelprotokoll".

Place a looping storm ambience file here named exactly:

    storm.mp3

It is entirely optional. The game is fully playable and solvable WITHOUT sound:
flash timing is driven by an internal clock and anticipated via the on-screen
visual metronome, never by audio. If this file is missing, the 🔊 toggle simply
does nothing.

No audio file is committed to keep the repo asset-free.
